# 安全 (通用)

## 输入校验
- 校验类型、长度、格式、范围
- 拒绝意外字段 (allowlist, 不是 denylist)
- 参数化查询 — 永远不用字符串拼接 SQL

## 密钥管理
- 所有 secrets 通过环境变量或密钥管理器
- 不提交 secrets — 用 `.env.example` 放占位符
- 不同环境用不同密钥

## 日志安全
- 记录安全事件: 认证失败、权限拒绝、异常行为
- 永不记录: 密码、token、完整信用卡号、PII
- 敏感数据脱敏: `user@***.com`, `tok_...abc`

## 依赖
- 优先用维护良好的知名包
- 定期检查已知漏洞
- lock 文件必须提交
